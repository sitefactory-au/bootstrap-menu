Bootstrap Context Menu

Submenu demo

Notes

  • Dropdown submenus were removed in Bootstrap 3 (see see SO post here), so unless you are using this with Bootstrap 2, you will need to include the extra CSS.
  • My fork refactors a lot of the original library's "actions" code, into "menuItems", which I feel is more consistent with the other default options, and easier to manage the menu/submenu structure.
  • The "actionGroups" option has been removed, in favor of adding "divider" items to the "menuItems" object.
  • The "fa-lg" class has been removed from icons. Set this in the "iconClass" property if you want larger icons.
Right click anywhere inside this box

New "menuItems" configuration:

var menu = new BootstrapMenu('#demo1Box', {
      menuItems: [{
        name: 'Submenu',
        iconClass: 'fa-bars',
        subMenuItems: [
          {
            name: 'Add',
            title: 'Add a new thing',
            iconClass: 'fa-plus',
            onClick: function() {
               toastr.info("'Add' clicked!");
            }
          },
          {
            name: 'Remove',
            title: 'Remove this thing',
            iconClass: 'fa-remove',
            onClick: function() {
               toastr.info("'Remove' clicked!");
            }
          }
        ]
      }, 
      {
        divider: true
      },
      {
        name: 'Another action',
        title: 'This will perform another action',
        onClick: function() {
          toastr.info("'Another action' clicked!");
        }
      }, 
      {
        name: 'A third action',
        title: 'This will perform a third action',
        onClick: function() {
          toastr.info("'A third action' clicked!");
        }
      }]
    });

Submenu CSS (for Bootstrap 3+):


.dropdown-submenu {
    position:relative;
}
.dropdown-submenu>.dropdown-menu {
    top:0;
    left:100%;
    margin-top:-6px;
    margin-left:-1px;
    -webkit-border-radius:0 6px 6px 6px;
    -moz-border-radius:0 6px 6px 6px;
    border-radius:0 6px 6px 6px;
}
.dropdown-submenu:hover>.dropdown-menu {
    display:block;
}
.dropdown-submenu>a:after {
    display:block;
    content:" ";
    float:right;
    width:0;
    height:0;
    border-color:transparent;
    border-style:solid;
    border-width:5px 0 5px 5px;
    border-left-color:#cccccc;
    margin-top:5px;
    margin-right:-10px;
}
.dropdown-submenu:hover>a:after {
    border-left-color:#ffffff;
}
.dropdown-submenu.pull-left {
    float:none;
}
.dropdown-submenu.pull-left>.dropdown-menu {
    left:-100%;
    margin-left:10px;
    -webkit-border-radius:6px 0 6px 6px;
    -moz-border-radius:6px 0 6px 6px;
    border-radius:6px 0 6px 6px;
}
